# 俄罗斯方块 - 需求设计文档

## 项目概述

一款支持多种游戏模式的网页版俄罗斯方块，包含**经典模式**、**竞速模式**、**节奏模式**和**变异模式**。前端 HTML/CSS/JS 实现，后端 Python Flask + SQLite 提供用户系统和数据持久化。

## 功能需求

### 1. 用户系统（FR-001）

| ID | 功能 | 说明 |
|----|------|------|
| FR-001-1 | 用户注册 | 用户名 + 密码注册，密码至少 6 位 |
| FR-001-2 | 用户登录 | 用户名 + 密码登录，支持会话保持 |
| FR-001-3 | 修改名称 | 登录后可修改显示名称 |
| FR-001-4 | 修改密码 | 登录后可修改登录密码 |
| FR-001-5 | 退出登录 | 清除本地会话 |

### 2. 设置系统（FR-002）

| ID | 功能 | 说明 |

| FR-002-3 | ARR 设置 | 自动重复移动间隔帧数（0-20 帧，默认 2） |
| FR-002-4 | 预览数量设置 | 下一个方块预览数量（1-8 个，默认 5） |
| FR-002-5 | 设置云端同步 | 登录后按键和 DAS/ARR 设置云端存储 |

### 6. 变异模式（FR-006）

| ID | 功能 | 说明 |
|----|------|------|

| FR-006-5 | 多选组合 | 上述变异效果可同时勾选组合游玩 |

### 7. 管理员后台（FR-007）

| ID | 功能 | 说明 |
|----|------|------|
| FR-007-1 | 管理员登录 | 独立端口 9047 的管理后台，管理员凭据登录（默认 admin / admin123456） |
| FR-007-2 | 排行榜查看 | 竞速40行排行榜，排除被封禁账号 |
| FR-007-3 | 账号注册情况 | 查看所有玩家：账号名、40行最佳记录（用时/分数）、DAS/ARR 设置、预览数量、方块颜色设置 |

| FR-007-7 | 同IP小号检测 | 同一IP登录不同账号时弹窗提示"游戏不提倡开小号"，确认则记录IP关联继续登录，取消则退出登录 |
